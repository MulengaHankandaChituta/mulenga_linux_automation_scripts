# Linux Automation Projects (Bash)

A collection of Linux automation scripts designed to simulate real-world system administration and database management tasks.

## Projects Included

### 1. Automated Backup Manager
- Compresses and timestamps backups
- Implements retention policy
- Logs activity
- Designed for production-style backup workflows

### 2. Restore Utility
- Restores compressed backups
- Validates execution success

### 3. System Health Monitor
- Tracks CPU, memory, and disk usage
- Logs results
- Implements alert threshold for disk usage

### 4. Database Backup Script
- Automates MySQL database backups using mysqldump
- Timestamped SQL dump generation
- Designed for DBA-style automation

### 5. Log Analyzer
- Parses authentication logs
- Extracts failed login attempts
- Displays top suspicious IP addresses

## Technologies Used
- Bash
- Linux CLI utilities (awk, grep, tar, find)
- MySQL tools (mysqldump)
- Cron-ready automation

## Use Case
These scripts demonstrate:
- System administration skills
- Database backup management
- Log parsing and analysis
- Automation of routine tasks
- Linux production environment familiarity
